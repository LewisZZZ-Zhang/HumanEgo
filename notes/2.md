Each `all_data/<idx>/` folder is one synchronized frame bundle: raw image, masks, overlays, Aria state, and the final per-frame training target.

Evidence: the pipeline writes `rgb.png` + `aria_cam_rgb.json` in [AriaCamTypes.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/AriaCamTypes.py:132), hands in [AriaHandsTypes.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/AriaHandsTypes.py:275), SLAM in [AriaSlamTypes.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/AriaSlamTypes.py:98), phases in [AriaPhasesTypes.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/AriaPhasesTypes.py:87), masks in [DINOSAM.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/DINOSAM.py:182), keypoint images in [VisualKpts.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/VisualKpts.py:310), and `training_data.json` in [DatasetGen.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/DatasetGen.py:407).

| File | Purpose |
|---|---|
| `rgb.png` | The extracted rectified RGB camera frame. Written by `save_aria_cam_json()` alongside camera metadata. |
| `aria_cam_rgb.json` | Per-frame camera metadata: `idx`, timestamp, image size, FPS, intrinsics `k`, distortion `d`, camera-to-world `c2w`, depth/camera transforms, and `rgb_path`. Used for projection, geometry, and dataloader metadata. |
| `aria_hands.json` | Per-frame MPS hand state for `hand_r` / `hand_l`: confidence, grasp state, wrist/palm/midpoint poses, 2D/3D keypoints, optimized world poses, and velocities. `DatasetGen` reads it to produce hand SE(3) state and grasp labels. |
| `aria_slam.json` | Per-frame headset/device motion from SLAM: world translation, roll/pitch/yaw, deltas, linear speed, angular speed, unwrapped yaw. This is motion context and also feeds phase estimation. |
| `aria_phases.json` | Per-frame task/temporal phase label: `stop`, numeric `mode`, `mode_str`, speeds, yaw. `DatasetGen` uses `mode == 4` to set `metadata.is_finished`. |
| `mask_obj1.png` | Segmentation mask for object prompt `obj1`. In `serve_bread`, `obj1` is `"piece of bread ."` from [serve_bread.yaml](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/cfg/preprocess/tasks/serve_bread.yaml:4). |
| `mask_obj2.png` | Segmentation mask for object prompt `obj2`. In `serve_bread`, `obj2` is `"a plate ."` from the same task config. |
| `mask_arm.png` | Segmentation mask for the human arm/hands prompt. Used by LaMa to inpaint/remove the arm and by `training_data.json` as `obs.mask_arm_path`. |
| `mask_arm_and_obj.png` | Combined OR mask of all DINOSAM masks: arm plus objects. Written after all prompt masks are processed in `DINOSAM.process_and_save()`. `DatasetGen` stores this as `obs.mask_obj_path`, despite the name meaning “combined arm+object mask.” |
| `rgb_WArmObjKpts.png` | Original RGB frame with hand/gripper keypoints and tracked object keypoints overlaid. `WArmObjKpts` means “with arm/object keypoints.” |
| `rgb_WoArm_WArmObjKpts.png` | Arm-inpainted visual input with the same hand/object keypoints overlaid. This is the default vision input for training: `IMG_NAME = "rgb_WoArm_WArmObjKpts.png"` in [FlowMatchingDataloader.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/training/FlowMatchingDataloader.py:51). |
| `training_data.json` | Final consolidated per-frame training sample: metadata, camera intrinsics/pose, image/mask paths, hand transforms/grasp, object transforms, dynamic flags, anchor frame, and finished flag. The dataloader indexes every `all_data/<idx>/training_data.json` it finds. |

One useful detail: `DatasetGen` explicitly requires the four JSON dependencies `aria_cam_rgb.json`, `aria_hands.json`, `aria_phases.json`, and `aria_slam.json` before writing `training_data.json` ([DatasetGen.py](/Users/lewis/SchoolWorks/06_Junior_Spring/research1/HumanEgo/preprocess/DatasetGen.py:411)). So those JSONs are the measured/intermediate state, while `training_data.json` is the cleaned training-facing bundle.
